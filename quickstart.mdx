Get up and running with the IBMR C Exercises repository in under 5 minutes.

## Prerequisites

Before you begin, make sure you have:

- A C compiler installed (GCC is recommended)
- Git installed
- A terminal (Linux, macOS, or Windows with WSL/Git Bash)

<Accordion title="How to install GCC">

<Tabs>
  <Tab title="Linux (Debian/Ubuntu)">
    ```bash
    sudo apt update
    sudo apt install build-essential
    gcc --version
    ```
  </Tab>
  <Tab title="macOS">
    ```bash
    xcode-select --install
    gcc --version
    ```
    Or install via Homebrew:
    ```bash
    brew install gcc
    ```
  </Tab>
  <Tab title="Windows">
    Install [MinGW-w64](https://www.mingw-w64.org/) or use **WSL** (Windows Subsystem for Linux).

    With WSL:
    ```bash
    sudo apt update && sudo apt install build-essential
    ```
  </Tab>
</Tabs>

</Accordion>

## Get the code

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/insxnsive/ibmr-c-exercises.git
    cd ibmr-c-exercises
    ```
  </Step>
  <Step title="Explore the files">
    ```bash
    ls
    ```
    You'll see the available C source files. Currently includes `rpg.c` — a fully commented RPG combat game.
  </Step>
  <Step title="Compile a program">
    Use GCC to compile a `.c` file into an executable:
    ```bash
    gcc rpg.c -o rpg
    ```
    This creates an executable called `rpg` (or `rpg.exe` on Windows).
  </Step>
  <Step title="Run the program">
    <Tabs>
      <Tab title="Linux / macOS">
        ```bash
        ./rpg
        ```
      </Tab>
      <Tab title="Windows">
        ```bash
        rpg.exe
        ```
      </Tab>
    </Tabs>
  </Step>
</Steps>

## Your first session

After running `rpg`, you'll see the game introduction and then a combat prompt:

```
Echoes mutter through your mind..
'Is this some sort of nightmare?' You ask yourself...
There's no point in going back now!!
========= GAUNTLET START! =========

>>> ENEMY #1 APPEARS! <<<

| --- YOUR TURN (Enemy 1/5) ---
| Your HP: 100 | Enemy HP: 25 | Potions: 3 |
| 1. Attack | 2. Guard | 3. Potion | 4. Gamble | 5. Flee
| - Your choice: 
```

Enter a number (1–5) to take your turn. Defeat all 5 enemies to win!

<Tip>
  Try compiling with warnings enabled for a better learning experience:
  ```bash
  gcc -Wall -Wextra rpg.c -o rpg
  ```
</Tip>

## Supported compilers and IDEs

<CodeGroup>

```bash GCC
gcc file.c -o output
./output
```

```bash Clang
clang file.c -o output
./output
```

</CodeGroup>

You can also use any C-capable IDE:

- **VS Code** — with the [C/C++ extension](https://marketplace.visualstudio.com/items?itemName=ms-vscode.cpptools)
- **Dev-C++** — Windows-friendly, beginner-friendly
- **Code::Blocks** — cross-platform, lightweight
- **CLion** — full-featured IDE from JetBrains
