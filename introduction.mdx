## What is IBMR C Exercises?

IBMR C Exercises is a collection of C programming exercises and examples created **by students, for students**. It was started by IT students at **IBMR** (Instituto Brasileiro de Mercado de Capitais) and is open to any student who wants to learn or practice the C programming language.

The repository is organized by topic, covering everything from writing your first `Hello, World!` program to managing memory with dynamic allocation and working with files. Each topic includes commented examples, hands-on exercises, and practical tips.

<Info>
  You don't need to be an IBMR student to use this repository. It is open to anyone who wants to learn C.
</Info>

## What each topic includes

Every topic in the repository follows the same structure to make learning consistent and approachable:

- **Commented examples** — source files with inline explanations so you understand what each line does
- **Exercises** — problems to solve on your own after studying the examples
- **Tips** — notes on best practices and common pitfalls in C programming

<Tip>
  If you are completely new to C, start with the **Hello World** topic and work through the list in order. Each topic builds on the concepts introduced before it.
</Tip>
